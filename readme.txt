Olá professora, aqui vão ter as instruções para que a senhora consiga abrir o jogo, ele foi feito usando html, css, javascript, php e mySQL
portanto, ele precisa ter um servidor local, usamos o heidiSQL e o xampp para fazer o servidor, e é recomendado que se use o Google Chrome para se jogar o jogo.

1º passo: entre na pasta de programas necessários e instale o Xampp, e o HeidiSQL nesta ordem, e se não tiver o Google chrome, baixe também.

2º passo: abra o Xampp e de "Start" no Apache e no SQL, para conseguir simular o servidor local.

3º passo: Entre na pasta Projeto final e vá para a pasta htdocs, você chega nela indo no disco local C:, entrando na pasta xampp, e depois na htdocs, podendo assim,
colar os arquivos e pastas que você copiou.

4º passo: entre na pasta banco de dados e clique duas vezes e vai abrir uma tela, nesta tela você só precisa clicar em abrir, ai vai ter um script na tela do MySql,
e após isto basta apertar f9, e seu banco de dados vai ser construído para utilizar.

5º passo: com o Xampp ligado e com a conexão feita no banco de dados,  a senhora está pronta para jogar o jogo.

6º passo: para abrir o jogo, a senhora precisa ir ate a barra de endereço do seu google chrome, e colocar este endereço http://127.0.0.1/ após isso só dar enter e vai abrir o jogo
